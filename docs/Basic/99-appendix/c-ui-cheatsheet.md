# C. UI 速查卡

这页不是设计理论课，只是给基础版够用的表达词。你在第 3 章改界面时，如果一时说不清，可以先来这里借词。

## 风格方向词

| 你想要什么感觉 | 可以这样说 |
|------|--------------|
| 简约专业 | 简洁、克制、信息优先、层次清楚 |
| 温暖友好 | 有人味、容易交流、轻松、不生硬 |
| 轻微科技感 | 清爽、数字感、未来感一点点、但不要炫技 |
| 更像作品 | 不像默认模板、有主次、有视觉重点 |

## 配色表达词

| 想表达什么 | 可以这样说 |
|------|--------------|
| 干净明亮 | 浅色、通透、清爽、留白多 |
| 稳重克制 | 低饱和、偏深、别太跳 |
| 有重点色 | 只保留一个主色，其余颜色收住 |
| 不想太花 | 不要过多渐变，不要颜色乱跳 |

## 排版与层次词

| 问题 | 可以这样说 |
|------|--------------|
| 页面太散 | 首屏信息太散，请加强主次关系 |
| 页面太挤 | 区块之间太挤，请增加留白和呼吸感 |
| 看不出重点 | 请让名字、一句话介绍和主按钮更突出 |
| 像信息平铺 | 请拉开标题、正文和辅助信息的层次 |

## 交互与体验词

| 目标 | 可以这样说 |
|------|--------------|
| 让访客知道你是谁 | 首屏打开后一眼就能知道我是谁 |
| 让聊天入口更容易被注意到 | 请让聊天入口更明显，但不要喧宾夺主 |
| 让按钮更像真的能点 | 请给主要按钮明确反馈和真实跳转 |
| 兼顾手机端 | 请确保手机端文字可读、区块不重叠 |

## 一个很够用的界面表达句式

```text
现在这个页面的问题是：______
我希望它变成这样的感觉：______
请先只调整这些部分：______
不要动这些部分：______
```

如果你能把“问题 + 感觉 + 边界”说清，通常就已经够用了。

# A. 常用提示词模板

这一页不讲理论，只收正文里最常用、也最值得反复复用的几类提示词。你不需要一字不改照抄，但当你一时不知道怎么开口时，它们会很有用。

## 1. 第一次生成作品模板

适用场景：第 1 章第一次把想法交给平台，让它先生成一个能预览的版本。

```text
我想做一个个人主页，上面有一个数字分身聊天区。

【关于我】
- 名字：______
- 一句话介绍：______
- 我现在主要在做：______
- 我的兴趣：______
- 我一个比较有记忆点的特点：______

【这个主页给谁看】
- 主要访客：______

【页面里至少要有】
- 头像
- 名字和一句话介绍
- 一个简单的个人信息展示区
- 一个可以提问的数字分身聊天区

【数字分身需要知道】
- 我的职业或身份：______
- 我最近在做的事：______
- 我擅长或关心的方向：______
- 别人最可能问我的 3 个问题：______

【风格要求】
- 我希望整体感觉是：______
- 我更偏好的颜色：______
- 是否要适配手机：是
- 其他要求：______

请先帮我生成一个第一版原型。
要求：
1. 先保证结构清楚、能用
2. 不要过度复杂
3. 先做出一个可预览的版本
```

## 2. 本地只改一处模板

适用场景：第 2 章以后，你已经把项目带回本地，只想改一句文案、一个按钮、一个小区块。

```text
我现在想修改这个项目里的一个小地方。

目标：______
请先告诉我这部分代码大概在哪个文件里。

要求：
1. 只改这一处
2. 不要动其他布局和样式
3. 改完后告诉我你改了什么
```

## 3. 界面优化模板

适用场景：第 3 章改首页风格、首屏排版、聊天区视觉层次。

```text
现在这个页面的主要问题是：______

我希望它改成这样的感觉：______

这次请你只修改这些地方：______
先不要动这些部分：______

另外请兼顾这些最基本的体验：
- 首屏一眼能看出我是谁
- 聊天入口更明显
- 手机端文字可读
- 不要让页面显得更乱
```

## 4. 添加内容模块模板

适用场景：第 4 章补作品区、联系方式、FAQ 或聊天引导。

```text
请帮我给个人主页补一个新模块。

模块名称：______
这个模块要解决的问题：______
希望访客看完之后知道什么：______

要求：
1. 保持当前页面整体风格
2. 不要一次新增太多模块
3. 文案先写到够用，不要过度铺陈
4. 如果涉及按钮或链接，请接成真实可点击状态
```

## 5. 数字分身说明书模板

适用场景：第 5 章让数字分身更像你，而不是泛泛而谈。

```text
你是我的数字分身，用来在个人主页里回答访客关于我的问题。

你的任务：
- 介绍我是谁
- 回答和我有关的问题
- 帮访客了解我最近在做什么、做过什么、怎么联系我

关于我：
- 我是：______
- 我最近在做：______
- 我擅长或长期关注：______

说话方式：
- 语气：______
- 回答尽量：简洁 / 真诚 / 人话一点 / 不装专家

边界：
- 不要编造我没做过的经历
- 不要假装知道我没提供的信息
- 不知道时要明确说不知道

下面是 1-3 组我的真实表达样本，请参考这些样本的表达习惯来回答。
```

## 用这些模板时，记住一件事

模板的价值不是“格式标准”，而是帮你把目标、范围和边界说清楚。写清楚这三件事，通常就已经比“帮我弄一下”强很多了。

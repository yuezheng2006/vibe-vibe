# 第 5 章：让数字分身更像你 —— 人设、回答与排错

到了这里，你的主页已经有了基本样子，也开始更完整了。但数字分身还有一件更核心的事要做：它不能只是“会回答”，而要开始“像你在回答”。这两者差别很大。前者只是一个功能存在，后者才是这个作品真正开始有代表性的时候。因为对个人主页来说，数字分身的价值从来都不在于它像百科全书一样什么都知道，而在于它能更稳定、更真实地代表你。

第 5 章要处理的，就是这件事。你会先建立一个够用的心智模型，知道访客的问题是怎么被送到模型那里、又怎么回到页面上的；然后你会开始写第一版“数字分身说明书”，把它知道什么、不知道什么、该怎么说话这些边界慢慢立起来；接着你会学会怎么看最常见的几类问题，知道不回复、答偏、风格不对、编造信息时，应该怎么排；同时也会顺手建立一点最小的成本与安全意识。

如果说前面几章更多是在做“外面那层”，第 5 章就是开始做“里面那层”。页面是不是像你，数字分身是不是像你，这两个问题不是同一个层次。后者更难一点，但也更能拉开差距。

## 这一章真正的变化是什么

前面几章更多是在改“壳”和“入口”，第 5 章开始往里走。它不会让你的项目突然变成复杂系统，但它会让数字分身真正开始承担代表你的工作。到这里，它不应该再只是一个挂在页面上的聊天组件，而更像一个被你调教过、知道边界、知道语气、知道什么时候该收住的数字分身。

这一章完成后，你不一定会拥有一个“特别聪明”的机器人，但你应该会拥有一个更像你、更稳、更可信的数字分身。这反而更重要，因为它才符合这个项目的真实目标。

# 基础篇

<BasicEditionUpdateBox />

这套基础篇已经按新版主线完成重写：围绕“个人主页 + 数字分身”这一个连续案例，带你从第一次生成，到本地修改、界面优化、内容补全、数字分身校准，再到正式上线。

## 这套基础篇在做什么

这不是一本“先学语法、再学框架、最后再做项目”的传统教材。

它更像一条可以直接走通的路线：

- 先做出一个能运行、能展示、能对话的作品
- 再把它带回自己的电脑，学会继续修改
- 再一步步把它改得更像一个真正的产品
- 最后正式上线，拿到别人也能访问的链接

这套基础篇默认你是 **独立 vibe coder**。你不需要假设自己身边有前端、后端、测试同事。你要学会的，是如何把想法说清楚，如何把 AI 当成执行伙伴，如何判断结果是不是对的。

## 你最终会做出什么

本教程的主线案例是：**个人主页 + 数字分身**。

| 部分 | 它负责什么 | 为什么适合基础版 |
|------|-------------|------------------|
| 个人主页 | 展示你是谁、你做过什么、别人如何联系你 | 结果直观，容易建立成就感 |
| 数字分身 | 代替你回答别人最常问的问题 | 天然适合引入 AI，对话结果也容易验证 |
| 正式发布 | 把作品变成稳定链接，分享给别人看 | 帮你完成第一次“从想法到上线”的闭环 |

## 章节结构（v2）

## 怎么使用这套教程

::: tip 推荐学习方式
1. **先动手，再回看**：如果你容易焦虑，先做出东西会更有信心。
2. **每章都要亲手操作**：这不是只靠阅读就能掌握的教程。
3. **遇到问题先问 AI**：本教程不会要求你先自己硬扛半小时。
4. **不用一次看完附录**：附录是“遇到再查”，不是“提前全背”。
:::

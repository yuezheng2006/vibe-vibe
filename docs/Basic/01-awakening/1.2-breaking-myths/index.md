# 1.2 用填空式模板写出第一版需求

## 为什么不用你从零写 prompt

对零基础读者来说，最常见的障碍不是“没有想法”，而是：

> 我脑子里有画面，但我不知道怎么把它说成一段像样的话。

所以这一节不要求你从零写 prompt，而是先用一个 **填空式模板** 起步。

它的作用不是让你写出“最专业的 prompt”，而是帮你把最重要的信息放到对的位置上。

## 模板里要交代哪几类信息

上一节你准备的是一批散的素材，这一节要做的，不是重新想一遍，而是把那些素材装进一个更容易被平台理解的结构里。

你先把它理解成五个问题：你想做什么，页面里最基本要出现什么，谁会来看，数字分身要知道什么，整体希望是什么感觉。只要这五类信息说清楚，第一版就已经很够用了。

## 直接可用的模板

把下面这份模板复制到你的笔记工具里，然后开始填空：

```text
我想做一个个人主页，上面有一个数字分身聊天区。

【关于我】
- 名字：______
- 一句话介绍：______
- 我现在主要在做：______
- 我的兴趣：______
- 我一个比较有记忆点的特点：______

【这个主页给谁看】
- 主要访客：______

【页面里至少要有】
- 头像
- 名字和一句话介绍
- 一个简单的个人信息展示区
- 一个可以提问的数字分身聊天区

【数字分身需要知道】
- 我的职业或身份：______
- 我最近在做的事：______
- 我擅长或关心的方向：______
- 别人最可能问我的 3 个问题：______

【风格要求】
- 我希望整体感觉是：______
- 我更偏好的颜色：______
- 是否要适配手机：是
- 其他要求：______

请先帮我生成一个第一版原型。
要求：
1. 先保证结构清楚、能用
2. 不要过度复杂
3. 先做出一个可预览的版本
```

## 一个填好的示例

下面这个例子，本质上就是把上一节那些散的事实，翻成了一段平台更容易执行的需求：

```text
我想做一个个人主页，上面有一个数字分身聊天区。

【关于我】
- 名字：林安
- 一句话介绍：一个正在学习用 AI 做产品的内容策划
- 我现在主要在做：整理自己的作品和写作方向
- 我的兴趣：AI 应用、写作、旅行
- 我一个比较有记忆点的特点：喜欢把复杂问题讲成人话

【这个主页给谁看】
- 主要访客：朋友、潜在合作伙伴、面试官

【页面里至少要有】
- 头像
- 名字和一句话介绍
- 一个简单的个人信息展示区
- 一个可以提问的数字分身聊天区

【数字分身需要知道】
- 我的职业或身份：内容策划
- 我最近在做的事：搭自己的个人主页，整理作品集
- 我擅长或关心的方向：内容表达、AI 应用、知识整理
- 别人最可能问我的 3 个问题：你现在在做什么？你有哪些作品？怎么联系你？

【风格要求】
- 我希望整体感觉是：简约、清爽、稍微有一点科技感
- 我更偏好的颜色：深蓝色和白色
- 是否要适配手机：是
- 其他要求：不要太花哨，信息层次要清楚

请先帮我生成一个第一版原型。
要求：
1. 先保证结构清楚、能用
2. 不要过度复杂
3. 先做出一个可预览的版本
```

## 这个模板为什么有效

因为它不是在追求“写得漂亮”，而是在帮你完成三件事：**补上下文**，让 AI 知道你是谁；**缩小范围**，让 AI 不要一下生成过多复杂功能；**明确优先级**，让第一版先求能预览、能展示、能聊天。

它最重要的价值，不是格式本身有多标准，而是能防止你在第一步就漏掉关键信息。你也可以把它理解成一个过渡层：上一节是你的原始素材，这一节是把素材整理成平台更容易吃进去的版本。

## 你现在要做的动作

不要继续往下看太久。先停下来，真的把模板填一版。

就算你觉得自己现在填得还不够好，也先完成第一版。因为下一节我们就要把它真正扔进平台，生成第一个版本了。

你不需要从零写 prompt，先用填空式模板起步就够了。它的作用很简单：把上下文补上，把范围收住，把第一版目标说清楚。写完之后，下一步就是把它交给平台，生成第一个版本。

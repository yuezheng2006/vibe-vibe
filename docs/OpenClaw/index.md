# OpenClaw 教程

> 从零开始，7 天掌握你的 AI 私人助理 | The open-source guide to OpenClaw

OpenClaw 是全球最热门的开源 AI 私人助理平台（150k+ ⭐）。本教程帮助你用 7 天时间，从零搭建一个 24 小时在线、了解你、能办事、会主动工作的 AI 助手。

## 核心能力

- **多渠道通信**：飞书、钉钉、企业微信……你用什么聊天工具，它就在那里
- **工具调用**：执行命令行、读写文件、上网搜索、操作浏览器、调用 API
- **技能系统（Skills）**：像手机装 App 一样，给助手安装新技能
- **记忆系统**：短期记忆、长期记忆、身份记忆
- **心跳机制**：不是你找它，是它定期醒来，检查有没有需要处理的事
- **完全本地部署**：所有数据都在你的机器上，不经过任何第三方

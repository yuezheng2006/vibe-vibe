## 摘要

该论文将 vibe coding 作为一项“研究探针”，用于探索基于 AI / 语音的编码方式如何帮助缺乏传统编程背景的研究者进行软件构建，关键词包括 AI/voice-based coding、vibe coding、researchers without coding skills。 作者首先介绍了 Karpathy 提出的 vibe coding 概念，并在实验中让非专业程序员通过语音和自然语言与代码生成系统交互，记录其在实现研究原型、数据处理脚本等任务时的体验与瓶颈。 研究显示，这种方式显著降低了入门门槛，但也暴露出表达精确需求、调试复杂错误以及理解生成代码的困难，因此作者建议未来工具在交互设计上增加可视化反馈、结构化参数配置和引导式调试辅助，以提高对学术用户群体的安全性和可靠性。

## 摘要

这篇文章是对 Vercel 自身 “State of Vibe Coding” 调研结果的解读摘要，核心关键词包括 State of Vibe Coding、full-stack AI apps、Next.js、visual & vibe coding、production readiness。 文中回顾 Karpathy 提出概念以来的采用曲线，并结合对 Vercel 用户群的调查结果，分析哪些类型的团队正在把 vibe coding 与前端框架、边缘计算和无服务器后端结合，构建“从 prompt 到部署”的流水线。 创新点在于将数据与产品定位结合：Vercel 一方面展示用户在实验项目和营销落地页中积极使用 AI 生成代码，另一方面坦言真正把 vibe coding 用在核心业务上的比例仍然有限，安全和性能顾虑显著。 报告未公开全部原始数据，但提供了若干关键比例和趋势判断，为理解“以 JavaScript/前端生态为中心”的 vibe coding 采用情况提供了重要样本，也为其自家工具路线（如 AI 集成、模板化项目）做了铺垫。

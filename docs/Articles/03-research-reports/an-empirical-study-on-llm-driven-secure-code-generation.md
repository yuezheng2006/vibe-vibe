## 摘要

这项研究全面评估了最先进的 LLM 在安全代码生成方面的表现。研究发现 LLM 生成的代码中往往存在安全漏洞，即使是最新的模型版本也无法完全避免安全问题。研究系统分析了不同 LLM 在处理常见安全漏洞时的表现。

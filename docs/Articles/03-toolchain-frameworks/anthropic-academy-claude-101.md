# Anthropic 官方：Claude 101 与学院课程索引

**课程定位（据官方描述）**：学习如何把 Claude 用于日常工作任务、理解核心功能，并了解通往更进阶主题的延伸资源。

## 学习内容纲要（Claude 101）

下列模块名称来自公开课程介绍与学员分享，**具体课节以 Skilljar 内实际目录为准**（官方会迭代更新）：

- 什么是 Claude、如何开启第一次对话
- 如何写出更可靠、更符合预期的提示与任务描述
- 桌面端：对话、Cowork、Code 等能力概览
- Projects（项目）入门：如何把材料与上下文组织在一起
- Artifacts：用结构化产出物协作创作
- Skills：让 Claude 在合适任务上自动套用可复用说明
- 连接你的工具（集成与扩展思路）
- 企业场景中的检索与知识发现（如适用）
- Research mode：适合做深读与长程调研时的用法
- 按角色划分的实践用例参考
- 其他协作方式与延伸学习入口
- 完成学习后可申请**结业证明**（以平台规则为准）

## 访问与账号（常见问题）

- **是否需要 Anthropic / claude.ai 账号才能上课**：学习托管在 Skilljar，**一般只需 Skilljar 账号**即可访问课程材料；若要在课后持续使用 Claude 产品，再另行注册 [claude.ai](https://claude.ai) 等。
- **数据与隐私**：平台会记录学习进度、课时完成度、测验分数等；详情见官方说明与 [Skilljar Privacy Policy](https://www.skilljar.com/privacy)。
- **学习数据删除**：可发邮件至 [academy-support@anthropic.com](mailto:academy-support@anthropic.com) 咨询删除学习数据（以官方回复与法规为准）。

## 摘要

这一期播客聚焦 “vibe coding” 作为新兴开发范式，主持人与嘉宾围绕如何通过自然语言与 AI 共同完成软件开发展开讨论，关键词包括 prompt-driven development、LLM coding assistants、developer experience。 对话中，嘉宾分享了实际团队在用 AI 工具进行前端、小型服务开发时的经验：从需求拆解、形成高层 PRD，到通过对话生成代码、再迭代指令修复 bug，并讨论在这个过程中开发者技能结构如何从“写代码”转向“设计意图与验证结果”。 节目的议题性在于，嘉宾指出当前工具在上下文管理、长程记忆和测试自动化方面仍有明显短板，如果缺乏工程规范，vibe coding 易被误用在关键生产系统，带来质量与安全隐患，因此他们主张将其首先应用在原型、内部工具和探索性项目中。​

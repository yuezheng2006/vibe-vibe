## 摘要

这期节目邀请 Swyx（Sean Wang）讨论 2025 年 AI 编码的转折，标题用 “RIP Vibe Coding” 表达观点：纯对话式写代码的热潮正在减退，行业正在转向更具自主性的 agentic coding 与 “Agent Labs”，核心关键词包括 vibe coding、agentic coding、AI code agents、software engineering future。 对话回顾了 2025 年上半年开发者围绕 vibe coding 的狂热与质疑，指出在实际工程中，单靠对话生成代码在可维护性、团队协作与 CI/CD 集成上暴露出大量问题，而以目标为中心的代理式开发（code agents）则更适合融入现有流水线。 创新点在于节目提出了一个中长期判断：未来的软件工程价值可能主要来自“code AGI”——能端到端完成规划、实现、测试与部署的系统，而 vibe coding 将被重新定位为更友好的上层接口，而非完整开发范式，这一观点在播客听众和工程社区引发了不少争论。​
